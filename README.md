# Scaty3D.js

Scaty3D.js, WebGL tabanlı, JavaScript ile yazılmış basit bir 3D motor / deneme projesidir.  
Amaç; tarayıcı üzerinde çalışan, hafif ve öğrenme odaklı bir 3D altyapı geliştirmektir.

Bu proje ticari bir motor olma iddiası taşımaz. Daha çok deneme, öğrenme ve açık kaynak geliştirme amacıyla oluşturulmuştur.

## Scaty3D.js Nedir?

Scaty3D.js;

- WebGL kullanır  
- JavaScript ile çalışır  
- Tarayıcı üzerinde 3D sahneler oluşturmayı hedefler  
- Küçük ölçekli oyunlar ve 3D denemeler için tasarlanmıştır  

Proje, Scaty çatısı altında başlatılmıştır ve geliştirme süreci **SWO (Scaty Web Organization)** tarafından yürütülmektedir.
